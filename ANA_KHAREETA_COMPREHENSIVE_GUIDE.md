# 🗺️ مشروع "أنا خريطة" - وثيقة شاملة

## 📋 نظرة عامة على المشروع

**أنا خريطة** هو تطبيق ويب تعليمي تفاعلي مصمم خصيصاً للأطفال في سن 10-11 سنة (الصف الخامس الابتدائي) لشرح **الوحدة الثالثة: كنوز مصر** من مادة الدراسات الاجتماعية - الفصل الدراسي الأول.

---

## 🎯 الأهداف الرئيسية

### 1. التعليمية
- ✅ تحويل المنهج من حفظ إلى فهم واستيعاب
- ✅ ربط المعلومة بالموقع الجغرافي بشكل تفاعلي
- ✅ تبسيط المفاهيم الصعبة بلغة الأطفال
- ✅ تعزيز التعلم الذاتي والاستكشاف

### 2. الترفيهية
- 🎮 جعل التعلم ممتعاً مثل الألعاب
- 🏆 نظام مكافآت وشارات محفز
- 🎨 رسومات وأنيميشن جذابة
- 💬 شخصيات كرتونية محببة

### 3. التقنية
- 📱 يعمل على الكمبيوتر والموبايل والتابلت
- 🚀 سريع وسلس
- 💾 حفظ تلقائي للتقدم
- 🔄 تحديث فوري

## 🎮 نظام اللعب والمكافآت

### 1. النقاط (Points/XP)

**طرق كسب النقاط:**
- 🗺️ زيارة موقع جديد: **+10 نقاط**
- ✅ إجابة صحيحة في الاختبار: **+20 نقطة**
- 🏅 كسب شارة جديدة: **+25 نقطة**
- 📚 إكمال درس كامل: **+100 نقطة**
- 🎯 إكمال اختبار بدون أخطاء: **+50 نقطة إضافية**

**النقاط التراكمية:**
- المجموع يظهر في الشاشة الرئيسية
- يحدد المستوى الحالي
- لا يمكن خسارة النقاط

### 2. المستويات (Levels)

**نظام التدرج (5 مستويات):**

**المستوى 1: مكتشف مبتدئ 🔍**
- النقاط المطلوبة: 0-100
- الوصف: "بداية رحلتك في اكتشاف كنوز مصر!"
- اللون: أخضر فاتح (#95E1D3)

**المستوى 2: مستكشف 🗺️**

- الوصف: "بدأت تعرف كنوز مصر!"
- اللون: أزرق فيروزي (#4ECDC4)

**المستوى 3: خبير كنوز ⭐**
- النقاط المطلوبة: 301-600
- الوصف: "أنت خبير حقيقي!"
- اللون: أصفر ذهبي (#FFE66D)

**المستوى 4: بطل مصر 🏆**
- النقاط المطلوبة: 601-1000
- الوصف: "أنت بطل! تعرف كل شيء!"

**المستوى 5: أسطورة كنوز مصر 👑**

### 3. الشارات (Badges)

**شارات تلقائية (Auto-unlock):**
1. 👣 **أول خطوة** - زيارة أول موقع

4. 🎓 **خبير** - إكمال درس واحد
5. 💯 **متقن** - اختبار كامل بدون أخطاء
6. 🏆 **بطل** - إكمال جميع الدروس

**شارات خاصة بالمواقع:**
7. 🪙 **عالِم الذهب** - زيارة منجم السكري + اختبار
8. 🔩 **مهندس حديد** - زيارة الواحات البحرية + اختبار
9. 🌱 **صديق الفلاح** - زيارة أبو طرطور + اختبار

11. 🌿 **صديق البيئة** - زيارة بنبان الشمسية + اختبار

13. 🚀 **بناء المستقبل** - إكمال درس المشروعات القومية

### 4. الأنيميشن والحركة

**مبادئ الأنيميشن:**
1. **سلس (Smooth)**: كل الحركات بـ `ease-out`
2. **سريع (Fast)**: 200-300ms
3. **طبيعي (Natural)**: استخدام `bounce` و `spring`

**أنواع الأنيميشن:**
```javascript
// دخول العناصر
initial={{ opacity: 0, y: 20 }}
animate={{ opacity: 1, y: 0 }}

// تأثير الضغط
whileTap={{ scale: 0.95 }}

// تأثير التمرير
whileHover={{ scale: 1.05 }}

// حركة مستمرة
animate={{ rotate: [0, 10, -10, 10, 0] }}
transition={{ repeat: Infinity }}
```

## 🎯 تجربة المستخدم (UX)

### 1. رحلة المستخدم (User Journey)

**الخطوة 1: الوصول**
```
المستخدم يفتح التطبيق
↓
شاشة ترحيب جميلة مع أنيميشن
↓
رمز خريطة متحرك 🗺️
↓
عنوان جذاب بألوان متدرجة
```

**الخطوة 2: التسجيل**
```
حقل إدخال الاسم
↓
"ما اسمك يا بطل؟ 👦👧"
↓
زر كبير ملون "ابدأ المغامرة! 🚀"
↓
حفظ الاسم في الـ Store
```

**الخطوة 3: الشاشة الرئيسية**
```
ترحيب شخصي: "أهلاً [الاسم]! 👋"
↓
عرض المستوى والنقاط الحالية
↓
شريط تقدم ملون ومتحرك
↓
3 أزرار كبيرة ملونة:
  - 🗺️ استكشف الخريطة (بنفسجي-وردي)
  - 📚 الدروس (أزرق-سماوي)
  - 🏆 جوائزي (أصفر-برتقالي)
↓
عرض آخر الشارات (إن وجدت)
```

**الخطوة 4: استكشاف الخريطة**
```
عنوان: "🗺️ خريطة كنوز مصر"
↓
تعليمات: "اضغط على أي مكان لتكتشف كنوزه!"
↓
Grid من المواقع (بطاقات ملونة)
↓
كل بطاقة:
  - رمز كبير (🪙⚡💧)
  - اسم الموقع
  - المحافظة
  - أنيميشن عند التمرير
```

**الخطوة 5: التفاعل مع موقع**
```
المستخدم يضغط على موقع
↓
+10 نقاط (إذا كان جديد)
↓
نافذة منبثقة (Dialog) جميلة
↓
محتوى النافذة:
  1. رمز كبير متحرك
  2. اسم الموقع
  3. حوار الشخصية (4-5 فقرات ملونة):
     - مقدمة ترحيبية
     - حقيقة ممتعة (هل تعلم؟)
     - شرح مبسط
     - الموقع الجغرافي
     - الأهمية
  4. معلومات مبسطة (4 بطاقات):
     - ❓ ما هو؟
     - 📍 أين؟
     - 🎯 لماذا؟
     - 📊 كم؟
  5. زرين:
     - "🎯 ابدأ الاختبار" (أساسي)
     - "إغلاق" (ثانوي)
```

**الخطوة 6: الاختبار**
```
شاشة الاختبار
↓
عرض:
  - رمز الموقع
  - اسم الموقع
  - مؤشر التقدم (نقاط ملونة)
↓
السؤال في صندوق ملون
↓
3 خيارات كبيرة وواضحة
↓
المستخدم يختار
↓
تأثير فوري:
  - إجابة صحيحة:
    * لون أخضر
    * confetti 🎉
    * "+20 نقطة"
    * "ممتاز!"
  - إجابة خاطئة:
    * لون أحمر فاتح
    * "💪 حاول مرة أخرى!"
    * عرض الإجابة الصحيحة
↓
الانتقال للسؤال التالي (2 ثانية)
↓
عند الانتهاء: العودة للخريطة
```

**الخطوة 7: الجوائز**
```
شاشة الجوائز
↓
عرض المستوى الحالي:
  - رمز كبير متحرك
  - اسم المستوى
  - وصف تحفيزي
  - مجموع النقاط (رقم كبير)
  - شريط تقدم للمستوى التالي
↓
إحصائيات (3 بطاقات):
  - 🗺️ عدد الأماكن المكتشفة
  - 🏅 عدد الشارات
  - ⭐ مجموع النقاط
↓
عرض جميع الشارات:
  - Grid ملون
  - كل شارة قابلة للضغط
  - عند الضغط: احتفال!
```

### 2. تفاعلات الإيماءات (Gestures)

- ✅ Tap: فتح/اختيار

- ✅ Pinch: (لو كانت خريطة حقيقية)

**على الكمبيوتر:**
- ✅ Click: فتح/اختيار
- ✅ Hover: تأثير تكبير
- ✅ ESC: إغلاق النافذة

### 3. التغذية الراجعة (Feedback)

**أنواع التغذية:**
1. **بصرية (Visual)**:
   - تغيير اللون
   - أنيميشن
   - Confetti

   - رسائل تشجيعية
   - شرح الأخطاء
   - تلميحات

3. **صوتية (Audio)** (اختياري):
   - صوت نجاح
   - صوت خطأ
   - موسيقى خلفية

**توقيت التغذية:**
- فورية (Immediate): <100ms
- سريعة (Fast): 200-500ms
- متوسطة (Medium): 500-1000ms

## 📱 الاستجابة للشاشات (Responsive Design)

### Breakpoints

```css
/* Mobile First Approach */

/* Mobile (Default) */
/* 320px - 767px */
.container {
  padding: 1rem;
  max-width: 100%;
}

/* Tablet */
@media (min-width: 768px) {
  .container {
    padding: 2rem;
    max-width: 768px;
  }
}

/* Desktop */
@media (min-width: 1024px) {
  .container {
    padding: 2rem;
    max-width: 1024px;
  }
}

/* Large Desktop */
@media (min-width: 1280px) {
  .container {
    max-width: 1280px;
  }
}
```

### تكييف الواجهة

**Mobile (Portrait):**
- 1 column layout
- أزرار كبيرة (full width)
- خطوط أصغر قليلاً
- مسافات أصغر

**Tablet:**
- 2 columns layout
- أزرار متوسطة
- خطوط قياسية
- مسافات متوسطة

**Desktop:**
- 3-4 columns layout
- أزرار معتدلة
- خطوط كبيرة
- مسافات واسعة
